установка стандартных параметров 
SetProperties(UnicodeString Selection, TForm *Form1)
 - Selection - секция ini файла
 - Form1     - имя формы


Параметр	По умолчанию	Описание

Caption		-		Название формы

FormStyle	0		Поверх всех окон (0 - нет, 1 - да)

AllScreen	0		На весь экран (0 - нет, 1 - да)

CenterX		50		центра окна относительно экрана в процентах
CenterY		50

FontSize	10		Размер шрифта

FontName	Arial		Шрифт

Icon		-		Иконка окна

FormHeight	-		Высота формы

FormWidth	-		Ширина формы
